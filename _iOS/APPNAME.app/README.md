# `/[APPNAME].app`

[![banner](/.internals/trademarks/banner_1200x100.svg)](#)

This base directory houses the iOS app bundle procured from Apple AppStore.

You **DEFINITELY MUST NOT** place or modify any files and folders manually here
for avoiding breakage. Let the installed apps handle it.

The content are **NOT** backed up by iTunes and iCloud. However, iTunes does
perform initial sync of any apps procured from the Apple AppStore.

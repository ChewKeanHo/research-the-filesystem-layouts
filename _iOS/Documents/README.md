# `Documents`

[![banner](/.internals/trademarks/banner_1200x100.svg)](#)

This directory houses all data exposed to the user like user-generated drawings,
texts, videos, audios, photos, etc.

App is allowed to create additional directory here. Any app **SHOULD NOT**
assume any file or directory and always perform safe query before use.

The contents are backed up by iTunes and iCloud.

# `/Users/[USERNAME]/Applications/[APPNAME].app/Contents`

[![banner](/.internals/trademarks/banner_1200x100.svg)](#)

This directory houses an app's base structure.

This is specific to MacOS software bundle as other Apple OSes (e.g. `iOS`,
`iPadOS`, `tvOS`, etc) are using the flat version (without this `Contents`
directory) structure.

# `/Applications/[APPNAME].app/Contents/Resources`

[![banner](/.internals/trademarks/banner_1200x100.svg)](#)

This directory houses the app's resources files. It can be anything from linked
libraries files, sound files, interpretable source codes, image files, icon
files, internationalization datasets, etc.

It is entirely upto the app developer and the XCode version used to structure
this directory.

This directory is the fused directory from UNIX-based OSes' `lib/`, `share/`,
and `src/` directories.

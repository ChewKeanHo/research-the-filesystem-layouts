# `/Applications`

[![banner](/.internals/trademarks/banner_1200x100.svg)](#)

This base directory houses all app installed by the system administrator or the
operating system (OS). App procured via App Stores by the user are housed here
automatically.

The `Utilities` sub-directory contains a subset of apps intended for use in
managing the local system.

This directory is part of the `local domain`.

Only admin-privileged (`wheel`) users can access this directory.

You **DEFINITELY MUST NOT** place or modify any files and folders here for
avoiding any breakage (e.g. signed signature).

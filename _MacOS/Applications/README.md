# `/Applications`

[![banner](/.internals/trademarks/banner_1200x100.svg)](#)

This directory houses app installed by the user or the OS. App Stores installed
procured by the user housed here automatically.

The `Utilities` sub-directory contains a subset of apps intended for use in
managing the local system.

This directory is part of the `local domain` only admin-privileged (`wheel`)
users can modify or delete.

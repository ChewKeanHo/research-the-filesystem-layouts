# `/System`

[![banner](/.internals/trademarks/banner_1200x100.svg)](#)

This directory houses the system resources required by MacOS to run. They are
provided by Apple and **MUST NOT** be modified.

This directory is part of the `system domain`.

Only Apple internal systems can access this directory.

You **DEFINITELY MUST NOT** place or modify any files and folders here for
avoiding breakage.

# `/System/Library/User Template`

[![banner](/.internals/trademarks/banner_1200x100.svg)](#)

This base directory houses all user home directory structure template.

This directory is part of the `system domain`.

Only admin-privileged (`wheel`) users can view the content. Only system-level
installed apps can create, modify or delete contents in this directory.

In FreeBSD, this directory is facilitated via `/etc/skel` directory.

In `Linux`-based OSes, this directory is facilitated via `/etc/skel` directory.

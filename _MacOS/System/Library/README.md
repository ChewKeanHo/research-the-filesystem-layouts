# `/System/Library`

[![banner](/.internals/trademarks/banner_1200x100.svg)](#)

This directory houses system-level, system-wide configurations and data files
(e.g. app libraries or internal system files).

**NO app or user** is allowed to alter anything here. Only Apple proprietary
system programs and applications can operate here.

This directory is part of the `system domain`.

Only admin-privileged (`wheel`) users can view the content. Only system-level
installed apps can create, modify or delete contents in this directory.

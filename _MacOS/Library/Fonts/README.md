# `/Library/Fonts`

[![banner](/.internals/trademarks/banner_1200x100.svg)](#)

This is the local system directory housing system-wide font data files for
extending the operating system's functionalities from *Full Catalogue* stage to
*Complete* stage.

Generally, you **SHOULD ONLY** place distributor's registered files here.


For MacOS, you **SHOULD** use `Font Book` or `Settings` to install the font
files. Only administrators (with `wheel` privilege) can place custom data here.

All files here are available to all users.




## Naming Conventions

[![banner](/.internals/trademarks/banner_1200x100.svg)](#)

It is a practice to house the font data files using non-conflicting font family
name as filename.

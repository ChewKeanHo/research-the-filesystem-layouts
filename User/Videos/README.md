# `Videos`

[![banner](/.internals/trademarks/banner_1200x100.svg)](#)

This directory houses all motion picture files. Hidden objects remain hidden
unless configured for visibility.




## OS Deployment

[![banner](/.internals/trademarks/banner_1200x100.svg)](#)

The following operating systems deploy this directory:

1. `FreeBSD`.
2. `Linux`-based OSes.
3. Apple's `MacOS` (called `Movies` instead).
4. Microsoft `Windows`.

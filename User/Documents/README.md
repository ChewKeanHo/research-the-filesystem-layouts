# `Documents`

[![banner](/.internals/trademarks/banner_1200x100.svg)](#)

This directory houses all files and directories intended for documentation and
regular operations. Hidden objects remain hidden unless configured for
visibility.




## OS Deployment

[![banner](/.internals/trademarks/banner_1200x100.svg)](#)

The following operating systems deploy this directory:

1. `FreeBSD`.
2. `Linux`-based OSes.
3. Apple's `MacOS`.
4. Microsoft `Windows`.
